# Analyse des ventes – Projet Data

## Objectif
Analyser les ventes d’une PME afin d’identifier des tendances et aider à la prise de décision.

## Données
Données de ventes sur 20 jours (CSV).

## Outils
- SQL (SQLite)
- Python (Pandas, Plotly)

## Analyses réalisées
- Chiffre d’affaires total
- Ventes par produit
- Ventes par région

## Visualisations
- Quantité vendue par produit
- Chiffre d’affaires par produit

## Résultat
Identification des produits les plus performants et de la répartition des ventes.
